# Кодекс поведения

Этот проект — открытое сообщество вокруг таксономии ролей подкастинга. Мы ожидаем уважительного общения в Issues, Pull Requests и обсуждениях.

Сайт: https://podcasttaxonomy.ru/

## Наши стандарты

Приветствуется:

- уважительная критика идей и формулировок;
- конструктивные предложения и вопросы;
- готовность уточнять и улучшать описания ролей.

Недопустимо:

- оскорбления, травлю, дискриминацию;
- угрозы и домогательства;
- спам, рекламу продуктов и шоу;
- публикацию чужих персональных данных без согласия;
- намеренную порчу данных или вредоносные PR.

## Область действия

Кодекс действует в этом репозитории GitHub и в связанных каналах проекта, где обсуждается таксономия.

## Сообщения о нарушениях

Сообщите maintainers репозитория через GitHub (Issue с пометкой о модерации **или** приватное сообщение владельцу аккаунта [Te1man](https://github.com/Te1man)).  
Если речь о безопасности данных — используйте приватный Security Advisory репозитория, когда он доступен.

## Последствия

В зависимости от тяжести и повторяемости: предупреждение, скрытие/закрытие обсуждения, временный или постоянный запрет на вклад.

## Атрибуция

Текст основан на идеях [Contributor Covenant](https://www.contributor-covenant.org/) и адаптирован для этого проекта.
